# Radar System Using Ultrasonic Sensor and Arduino

## About
A 180-degree scanning radar prototype that detects objects in real-time
and visualises them on a radar sweep display.

## Technologies Used
- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- Servo Motor (SG90)
- Processing IDE
- Arduino IDE

## Features
- 180-degree object scanning
- Real-time radar sweep visualisation
- Object distance detection
- Live display using Processing IDE

## Circuit Components
- Arduino Uno board
- HC-SR04 ultrasonic sensor
- SG90 servo motor
- Jumper wires and breadboard
